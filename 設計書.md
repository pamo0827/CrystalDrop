# 雨アラート天気アプリ 設計書

## 1. アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 雨アラート（仮） |
| プラットフォーム | iOS |
| 対象ユーザー | 高齢者を含む全年齢層 |
| 主な機能 | 現在地取得・雨予報通知・シンプル表示 |
| 配布 | App Store（登録済み） |
| 追加費用 | なし（無料APIのみ使用） |

---

## 2. 技術スタック

| レイヤー | 技術 | 理由 |
|----------|------|------|
| 言語 | Swift 5.9+ | iOS標準・安定 |
| UI | SwiftUI | 宣言的UI・モダン |
| 位置情報 | CoreLocation | iOS標準・無料 |
| 天気API | **Open-Meteo** | 完全無料・APIキー不要・バックエンド不要 |
| 通知 | UserNotifications | iOS標準・無料 |
| データ永続化 | UserDefaults | 設定保存のみ・シンプル |
| バックエンド | **なし** | コスト0円 |

### Open-Meteo を選択する理由

- 完全無料・クレジットカード不要
- APIキー・アカウント登録不要
- アプリから直接HTTPS呼び出し可能
- 1時間ごとの降水確率・降水量データ取得可能

> ※ Apple WeatherKit（Apple Developer Programに含まれる）も選択肢だが、月50万回の上限管理が必要。Open-Meteoは上限なし（非商用利用）。

## 5. 機能仕様

### 5-1. 位置情報取得フロー

```
起動
  ↓
位置情報の許可を確認
  ├─ 未許可 → 許可を求めるダイアログ → 許可 → 位置取得
  │                                   → 拒否 → エラーメッセージ表示
  └─ 許可済み → 位置取得（精度: kCLLocationAccuracyKilometer）
  ↓
市区町村名を逆ジオコーディング（CLGeocoder）
  ↓
緯度経度をOpen-Meteo APIへ送信
```

### 5-2. 天気データ取得

**APIエンドポイント:**

```
GET https://api.open-meteo.com/v1/forecast
  ?latitude=35.6894
  &longitude=139.6917
  &hourly=precipitation_probability,precipitation
  &timezone=Asia/Tokyo
  &forecast_days=2
```

**取得データ:**

| フィールド | 内容 |
|-----------|------|
| `precipitation_probability` | 時間ごとの降水確率（%） |
| `precipitation` | 時間ごとの降水量（mm） |

**判定ロジック:**

```
当日 6:00〜22:00 の各時間の降水確率を取得
  ↓
最大値で状態を決定
  ├─ 0〜20%  → 晴れ（雨の心配なし）
  ├─ 20〜50% → 曇り（折りたたみ傘推奨）
  ├─ 50〜80% → 雨（傘を持って）
  └─ 80〜100% → 大雨（必ず傘を）
```

### 5-3. 通知機能

```
毎朝設定時刻（デフォルト 7:00）
  ↓
バックグラウンドで天気取得（BGAppRefreshTask）
  ↓
当日に降水確率 >= 50% の時間帯がある場合
  → 「今日は雨の予報があります。傘をお忘れなく ☔」通知
```

## 6. ディレクトリ構成

```
RainAlert/
├── App/
│   └── RainAlertApp.swift          # エントリポイント・通知設定
├── Views/
│   ├── HomeView.swift              # ホーム画面
│   ├── SettingsView.swift          # 設定画面
│   └── Components/
│       ├── WeatherStatusCard.swift # 状態カード（アイコン・メッセージ）
│       └── HourlyForecastRow.swift # 時間別予報（●○表示）
├── Models/
│   ├── WeatherData.swift           # 天気データ構造体
│   └── WeatherCondition.swift      # 状態enum（晴れ/曇り/雨/大雨）
├── Services/
│   ├── LocationService.swift       # 位置情報管理
│   ├── WeatherService.swift        # Open-Meteo API通信
│   └── NotificationService.swift  # 朝の通知スケジュール管理
└── Resources/
    └── Info.plist                  # 位置情報・通知の権限記述
```

## 7. Info.plist 必須キー

```xml
<!-- 位置情報（使用中のみ） -->
<key>NSLocationWhenInUseUsageDescription</key>
<string>現在地の天気を取得するために使用します</string>

<!-- 位置情報（バックグラウンド通知用） -->
<key>NSLocationAlwaysAndWhenInUseUsageDescription</key>
<string>毎朝の雨予報通知のために使用します</string>
```
